# MD Reports 1.0.0

Приложение для просмотра, редактирования, печати и экспорта Markdown-отчётов в PDF.

## Что скачать

- **`MD-Reports-Setup-1.0.0.exe`** — рекомендуемая версия для большинства пользователей. Обычный мастер установки Windows, ярлыки, привязка `.md`, автоматическая проверка WebView2.
- **`MD-Reports-Portable-1.0.0.zip`** — переносная версия без установки. Распакуйте архив и запустите `MD-Reports.exe`.

## Возможности

- просмотр Markdown с аккуратным оформлением;
- редактирование и сохранение `.md`;
- режимы «Просмотр», «Разделённый» и «Редактор»;
- экспорт в PDF;
- печать;
- светлая и тёмная темы;
- список недавних документов;
- открытие `.md` двойным кликом после привязки приложения.

## Системные требования

- Windows 10 или Windows 11 x64;
- WebView2 Runtime. В Setup-версии компонент устанавливается автоматически при отсутствии.
