# MD Отчёты 1.2.1

Исправлен формат поставки: основным файлом релиза снова стал Windows-установщик.

## Новое

- добавлен `MD-Reports-Setup-1.2.1.exe`;
- установщик ставит приложение в профиль пользователя Windows;
- новая установка переустанавливает предыдущую Electron-версию поверх старой;
- регистрируется открытие `.md`, `.markdown` и `.txt` через MD Reports;
- создаются ярлыки в меню «Пуск» и на рабочем столе;
- portable-версия оставлена отдельным файлом только для запуска без установки.

## Что скачать

- **`MD-Reports-Setup-1.2.1.exe`** - основной вариант для клиентов, установки, обновления и открытия `.md` двойным кликом.
- **`MD-Reports-Portable-1.2.1.exe`** - запуск без установки; не обновляет установленную версию и не регистрирует расширения.

## Проверено

- сборка Electron installer и portable;
- открытие Markdown-файлов через аргумент командной строки;
- smoke-тест PDF-экспорта;
- проверка JavaScript-файлов на синтаксис.
