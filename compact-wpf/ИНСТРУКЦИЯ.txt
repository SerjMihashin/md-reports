MD REPORTS
==========

Это компактная Windows-версия программы для просмотра и редактирования Markdown-отчётов.

ПЕРВЫЙ ЗАПУСК
-------------
1. Запустите MD-Reports.exe.
2. Нажмите «Открыть» и выберите файл с расширением .md.
3. Файл можно также перетащить мышью прямо в окно программы.

ЧТО УМЕЕТ ПРОГРАММА
-------------------
- красиво отображать Markdown-отчёты;
- редактировать и сохранять исходный .md-файл;
- показывать одновременно текст и готовый документ;
- сохранять оформленный отчёт в PDF;
- печатать документ;
- запоминать недавно открытые файлы;
- переключать светлую и тёмную темы.

КАК СВЯЗАТЬ ФАЙЛЫ .MD С ПРОГРАММОЙ
----------------------------------
Нажмите кнопку «Связать .md» в верхней панели.
После этого Windows зарегистрирует программу для файлов .md и .markdown текущего пользователя.

Если Windows ранее запомнила другую программу, один раз нажмите правой кнопкой мыши по .md-файлу:
«Открыть с помощью» → «Выбрать другое приложение» → MD-Reports.exe.

СИСТЕМНЫЕ ТРЕБОВАНИЯ
--------------------
- Windows 10 или Windows 11 x64;
- Microsoft .NET Desktop Runtime 8 x64:
  https://dotnet.microsoft.com/en-us/download/dotnet/8.0
- Microsoft Edge WebView2 Runtime:
  https://developer.microsoft.com/microsoft-edge/webview2/

На большинстве современных компьютеров WebView2 уже установлен вместе с Microsoft Edge.
