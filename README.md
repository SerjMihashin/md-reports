# MD Отчёты

Windows-приложение для просмотра, редактирования и печати Markdown-отчётов.

В репозитории сохранены две версии:

- стабильная Electron Portable `1.1.0`;
- компактная C# WPF + WebView2 версия в папке `compact-wpf`.

## Возможности

- открытие `.md`, `.markdown` и `.txt`;
- drag-and-drop;
- режимы «Просмотр», «Разделённый» и «Редактор»;
- сохранение файлов в UTF-8;
- список недавних документов;
- светлая и тёмная темы;
- прямой экспорт оформленного отчёта в PDF;
- печать через стандартное окно Windows;
- безопасный предпросмотр: HTML очищается перед отображением.

## Для клиента

1. Запустите `MD-Reports-Portable-1.1.0.exe`.
2. Нажмите «Открыть» или перетащите `.md`-файл в окно.
3. Для изменения текста выберите режим «Редактор» или «Разделённый».
4. Для сохранения нажмите «Сохранить».
5. Для PDF нажмите «Экспорт PDF» и выберите папку сохранения.

## Для сборки

```powershell
npm install
npm run build
```

Готовый portable-файл появится в папке `dist`.
