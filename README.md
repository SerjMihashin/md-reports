# MD Отчёты

Windows-приложение для просмотра, редактирования, печати и экспорта Markdown-отчётов.

## Что скачивать

Для обычной установки используйте **`MD-Reports-Setup-*.exe`** из GitHub Releases.

Установщик:

- устанавливает приложение в профиль пользователя Windows;
- переустанавливает приложение поверх предыдущей Electron-версии;
- создаёт ярлыки в меню «Пуск» и на рабочем столе;
- регистрирует открытие `.md`, `.markdown` и `.txt` через MD Reports;
- сохраняет пользовательские данные и список недавних документов при обновлении.

`MD-Reports-Portable-*.exe` нужен только для запуска без установки. Он не обновляет установленную версию и не регистрирует расширения файлов.

## Возможности

- открытие `.md`, `.markdown` и `.txt`;
- drag-and-drop файлов в окно приложения;
- режимы «Просмотр», «Разделённый» и «Редактор»;
- сохранение Markdown в UTF-8;
- список недавних документов;
- выбор нескольких недавних документов;
- удаление выбранных документов из истории без удаления файлов с диска;
- экспорт выбранных документов в ZIP;
- печать выбранных документов;
- светлая и тёмная темы;
- открытие веб-ссылок в браузере Windows по умолчанию;
- копирование блоков кода и отдельных inline-команд;
- экспорт текущего отчёта в PDF.

## Для клиента

1. Скачайте и запустите `MD-Reports-Setup-*.exe`.
2. Если Windows спросит про замену установленной версии, подтвердите установку.
3. Открывайте Markdown-файлы двойным кликом или через кнопку «Открыть».
4. Для изменения текста выберите «Редактор» или «Разделённый».
5. Для PDF нажмите «Экспорт PDF».

## Для сборки

```powershell
npm install
npm run build
```

Готовые файлы появятся в папке `dist`:

- `MD-Reports-Setup-<version>.exe` - установщик для клиентов;
- `MD-Reports-Portable-<version>.exe` - portable-версия без установки.
