# MD Отчёты 1.2.2

Исправлен переход со старой установленной версии на новый Electron-релиз.

## Новое

- добавлен `MD-Reports-Setup-1.2.2.exe` как основной файл для установки и обновления;
- установщик перед установкой ищет старую Inno Setup-версию `MD Reports` и удаляет её в тихом режиме;
- после миграции ставится новая Electron-версия;
- регистрируется открытие `.md`, `.markdown` и `.txt` через MD Reports;
- создаются ярлыки в меню «Пуск» и на рабочем столе;
- portable-версия остаётся отдельным файлом только для запуска без установки.

## Что скачать

- **`MD-Reports-Setup-1.2.2.exe`** - основной вариант для клиентов, обновления старой установки и открытия `.md` двойным кликом.
- **`MD-Reports-Portable-1.2.2.exe`** - запуск без установки; не обновляет установленную версию и не регистрирует расширения.

## Проверено

- сборка установщика и portable-файла;
- smoke-тест PDF-экспорта;
- проверка JavaScript-файлов на синтаксис;
- `latest.yml` указывает на установщик как основной файл.
