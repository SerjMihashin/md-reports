# MD Отчёты 1.2.0

Крупное обновление интерфейса и работы с несколькими Markdown-документами.

## Новое

- современный Fluent / Soft Productivity интерфейс;
- переработанные светлая и тёмная темы с поддержкой системной темы Windows;
- сворачиваемая боковая панель с фиксированной компоновкой;
- множественный выбор недавних документов;
- выбор всех документов, удаление из истории и экспорт выбранных файлов в ZIP;
- печать нескольких выбранных документов;
- открытие веб-ссылок в браузере Windows по умолчанию;
- копирование блоков кода одной кнопкой;
- копирование отдельных inline-команд кликом.

## Исправлено

- боковая панель и кнопка сворачивания больше не смещают заголовок документа;
- недавние документы располагаются сверху и не растягиваются по высоте панели;
- верхняя панель, строка документа и sidebar остаются фиксированными при прокрутке;
- исходный текст блоков кода копируется без служебных элементов и изменения переносов;
- улучшены границы, контраст и горизонтальная прокрутка блоков кода.
