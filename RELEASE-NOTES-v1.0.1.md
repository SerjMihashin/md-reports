# MD Reports 1.0.1

Небольшое обновление интерфейса.

## Исправлено

- тёмная тема теперь применяется ко всей оболочке приложения;
- боковая панель, строка текущего файла, рамки, подписи и кнопки режимов больше не остаются светлыми;
- сохранены все возможности версии `1.0.0`.

## Что скачать

- **`MD-Reports-Setup-1.0.1.exe`** — рекомендуемая версия для большинства пользователей;
- **`MD-Reports-Portable-1.0.1.zip`** — переносная версия без установки.
